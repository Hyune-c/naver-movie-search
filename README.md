# 네이버 영화 검색 API 로 어플리케이션을 만들어보자.

지금까지 학습하고 경험한 것들을 토이 프로젝트를 만들고 있습니다.  
주로 회사 업무 제약상 시도하지 못했던 기능이나 구현되어 있던 기능을 분석하는 것 위주로 간결하게 개발하였습니다.

### 개요

```
좀더 자세한 내용은 Wiki 에 정리되어 있습니다.
```

- 네이버 영화 검색 API 를 통한 영화 정보 DB 기록리

- 세션, 예외 처리
